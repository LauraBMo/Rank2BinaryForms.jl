Rank2BinaryForms.get_ratios — Method
get_ratios(u, M)

Given a vector u being a linear combination of the columns of M, returns the coefficients of such a linear combination. That is, returns v=[a_1,..,a_n] such that Mv = u.

``julia n = 100 v1 = matrix(QQ, rand(-n:n, 4, 1)) v2 = matrix(QQ, rand(-n:n, 4, 1))

u = 2v1-4v2 get_ratios([v1,v2], u) [ 2] [-4] ```

Rank2BinaryForms.set_getting_rootsofunity — Method
set_getting_rootsofunity(f)

Sets the method used to compute the d-th roots of unity. The input is a function f computing the roots of unity accepting calls as f(d)(k) and returning the k-th root of the d-th roots of unity. The order of the d-th roots of unity can be clockwise or anti-clockwise, but they must be ordered. Examples of valid inputs are the following functions f and g:

    g(d, k) = exp(2*pi*im*k/d)
    set_getting_rootsofunity(g)
    using Nemo
    CC = ComplexField(64);
    f(d, k) = Nemo.root_of_unity(CC, d)^k;
    set_getting_rootsofunity(f)
